Though you'll not find any tools here that actually change glyphs' outlines, you may find some helper tools here. Mind: none of these works on OpenType fonts with Postscript outlines; however, some may be changed easily to enable this, I think. Please do email me if you've got any plans in that direction: I may be able to help you on your way a bit.

All these I have made primarily for my own use, but they may be useful for you as well. They are all released under the GNU Public license. Any questions about the contents of the site or about the applications should be directed to me.

  • TrueTypeViewer enables you to view TrueType fonts, and, specifically, to debug the instructions embedded in the font. It has a "Features" button that will allows you to apply OpenType features to the string;
  • TTIComp compiles a .TTI file (with TrueType Instructions in a format not unlike C code) and a TrueType font into an instructed TrueType font;
  • OTComp takes a feature file in a format not unlike Adobe's and produces an OpenType font with advanced layout tables.
  • OTLegacy takes an OpenType font and adds Unicode precomposed characters to it by applying the OpenType features.

The sources for all these are also available. These work on Visual C++ 6.0 and gcc, but should compile on any C++ compiler with only a few changes. If you have made some changes you think I ought to apply, please email them to me. The versions of the programs posted now are brand new and have been totally revised

You might also want to take a look at the fonts I made so far.

A word of warning

Both the TrueTypeViewer and TTIComp use TrueType instructions, crucial parts of which are patented by Apple. One page explaining something about this is the one about the patents on the FreeType site (by the way, I'm not using the FreeType instruction engine in my code, but the same patent related problems apply to my own Virtual Machine). Though I am not a lawyer, what I understand is that you are not allowed to download or use those programs if you are in a country where those patents are valid (e.g., the USA) unless you have acquired a license from Apple.
